# iBizEHR 前端

基于 iBiz 体系构建的 iBizEHR，包含招聘管理、考勤管理、组织管理、人事管理、薪资管理、绩效管理等模块。

## 子项目说明

| 项目名称   | 功能描述                                                                                 |
| ---------- |--------------------------------------------------------------------------------------|
| `组织管理` | 多层级组织树、机构/部门/员工定义。[查看详细说明](base/README.md)                                           |
| `招聘管理` | 覆盖社招、校招、内推、猎头管理等场景，让 HR 获得更高效的招聘体验、更便捷的协作体验，让管理者获得招聘数据洞见。[查看详细说明](recruit/README.md) |
| `考勤管理` | 排班与排班模板：支持复杂班次与轮班规则。[查看详细说明](attendance/README.md)                                   |                                                                                                                                      |
| `人事管理` | `进行中`                                                                                |
| `薪资管理` | `规划中`                                                                                |
| `绩效管理` | `社区共建`                                                                               |
